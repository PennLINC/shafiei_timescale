# Mapping developmental patterns of intrinsic timescale

This repository contains code and documentation in support of the manuscript titled, ["Mapping developmental patterns of intrinsic timescale"](https://doi.org/10.64898/2026.04.08.717312), now available on bioRxiv.

## `code`
The [code](code/) folder contains all the code used to run the analyses and generate the figures.

Full reproducibility guide is available at [https://pennlinc.github.io/shafiei_timescale](https://pennlinc.github.io/shafiei_timescale).
